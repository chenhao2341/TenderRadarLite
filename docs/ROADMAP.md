# Roadmap

这份路线图描述的是当前务实演进方向，不代表全国稳定覆盖、自动投标或成熟 SaaS 承诺。

## v0.1-alpha

目标边界：

- 本地 HTML 报告
- 双 `supported` 来源
- 默认关闭的 `alpha` 异构来源
- Source Catalog
- 字段质量审计
- Windows 本地运行
- 安全边界说明

## v0.2-alpha

建议重点：

- Web 控制台真实运行入口
- 运行历史
- 来源启停可视化
- 错误日志展示
- 报告入口整合

## v0.3-beta

建议重点：

- Source Probe / 输入 URL 自动识别
- 更多异构来源
- 旧来源质量持续修复
- 结果公告摘要优化
- 配置界面完善
- 长期运行稳定性

## 当前明确不承诺

- 全国稳定覆盖
- 自动投标
- SaaS
- 登录 / 验证码绕过
- 附件深度解析

## 路线图说明

- `v0.1-alpha` 的核心是把“本地主路径 + 来源边界 + 安全边界 + 已知限制”讲清楚。
- `v0.2-alpha` 的核心不是继续堆来源，而是补足 Web 控制台真实运行入口。
- `v0.3-beta` 才适合把更多精力放到来源扩展和长期运行质量上。
