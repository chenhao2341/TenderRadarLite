# Roadmap

下面的路线图用于描述项目演进方向，不代表全国稳定平台承诺。

## 已完成阶段

- `v0.1` local HTML report
- `v0.2` optional Feishu integration
- `v0.3` Windows bat launcher
- `v0.4` adapter framework
- `v0.5` industry profiles

## 建议后续方向

- `v0.6` optional AI analysis alpha
- `v0.7` multi-source alpha
- `v0.8` Windows package / EXE experiment

## 路线图说明

- `v0.6` 如继续推进，应保持可选、alpha、非核心依赖定位。
- `v0.7` 重点是多来源 alpha 验证，而不是直接宣称全国覆盖。
- `v0.8` 只是 Windows 包装层实验，不代表本轮已经提供 EXE 或安装包。

## 当前务实定位

TenderRadarLite 目前的价值在于：

- 本地优先
- 结构清晰
- 可逐步扩展来源和行业规则
- 可以在不依赖飞书的情况下直接产出可读结果
